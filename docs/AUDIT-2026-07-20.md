## 6. 关键路径（行为可预期）

### 6.1 冷启动

```text
Codex.lnk / FastLaunch
  → open-codex-dream-skin.ps1
  → launcher-ui（安静 UI / 托盘 / 焦点）
  → current.json → versions/<id>/scripts/injector.mjs --watch
  → 发现/启动 Codex（CDP 9335）
  → wait-shell → attach browserId
  → control-plane :9336
  → 读 active-theme → 注入 CSS/JS
  → 写 state.json
```

### 6.2 热换肤

```text
cli apply --theme <id>
  → themes.writeActiveThemeFromHeige
  → core.kickThemeInjectNow → POST http://127.0.0.1:9336/kick
  → watch 重 apply（目标 ~45–80ms）
  → 控制面不可达时：spawn injector --once 降级（见 §9.2）
```

### 6.3 发布 vs 产品包

| 路径 | 写 git tree？ | 版本来源 | 产物 |
|------|:-------------:|----------|------|
| `publish-runtime.ps1 -Version` | **是**（唯一） | 参数 | 安装态 versions + stamp 源文件 |
| `Build-ProductPackage.ps1` | 否 | `-Version` 或已 stamp token；缺则 throw | `dist/CodexDreamSkin-<ver>-win-x64.zip` |
| `Install-Product.ps1` | 否 | meta → VERSION → token；缺则 throw | 安装树 + soft reattach + GC |
